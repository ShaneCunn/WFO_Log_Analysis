Introduction

The ACR logging tool will allow you to parse large log file by using a search string, this will extract nth line above or below or using the timestamps. This will write the parsed lines into a new log file.

Overview of the user interface


1. File Menu

  • Access to the About ACR logging Tool
  • Access to help and resource on ACR Logging Tool
  • Exit the application

2. Source & Output

  • Access to the choose file dialog
  • Access to the choose output location
  • Extra options: Time range filter and Exclude filter
  • Export group will export a currently selected group as a text file that can be import on an another computer
  • Import group can import a text file

3. Select Group

  • View of Select group
  • Run groups of rules when a group is ticked
  • Stop groups of rules when a search is running
  • Delete Group
  • Copy to new group

4. Current Lines

  • Used to edit the Select group of rules
  • Access to frequently used actions such as "Paste", "Copy", "Delete", and "Diagram Options"

5. Editor

  • Used to select and edit the Group name
  • Add New a group of rules
  • Save the current set of rules

Installing and Running ACR Logging Tool

You can modify the installation location and other settings using the ACR Logging Tool setup program


Note: The Windows user account used to install ACR Logging Tool must have administrator rights


To install ACR Logging Tool:
    1. Run the setup program (ACR Logging Tool.msi)
    2. Complete the setup wizard, providing the appropriate information as necessary


To run ACR Logging Tool:

Start ACR Logging Tool from the Avaya program group

Getting Started

Get started with ACR Logging Tool by selecting a source log file

ACR Logging Tool



To Search a Log after launching the tool for the first time:


  1. Click Choose File and select a log file to parse
  2. Click Choose Output and select where the parsed log file will output to

      Note: it will use the source log directory as a default for the output

  1. Click Add New
  2. Give the Group a name and Click Save
  3. Click on Search String box and enter the string to search for

      Note: By default it will use timestamps to do a search. Un-tick timestamps if you wish to do nth lines above or below

  1. Click Save
  2. Tick the script you wish to parse within the Select Script to Parse with box
  3. Click Run


Note: The length of time to pares a log may vary depending on the search string
